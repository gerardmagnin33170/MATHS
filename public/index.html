Progression IA - Élèves

Outil d'explicitation d'une progression des compétences IA dans le secondaire
Actions

Importer : Charger un fichier csv contenant des cartes.
Le format doit être conforme à celui utilisé pour l'export.

Exporter : Enregistrer les cartes actuelles dans un fichier csv.
Ce fichier peut être réimporté plus tard pour restaurer l'état des cartes.
