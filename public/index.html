Progression IA - Élèves

Outil d'explicitation d'une progression des compétences IA dans le secondaire
Réinitialiser la session
Pour repartir de zéro, comme si de rien n'était.
Sauvegarder la session
Sauvegarde au format json le travail effectué dans l'outil.
Charger la session
Permet d'importer le contenu du travail effectué dans une session précédente.
Exporter les cartes
Enregistrer les cartes actuelles dans un fichier csv.
Ce fichier peut être réimporté plus tard pour restaurer des cartes qui ont été supprimées ou modifiées.
Importer un fichier de cartes
Charger un fichier csv contenant des définitions de cartes.
Le format est le suivant :

id,categorie,"texte",niveauMin

Pour commencer, vous pouvez exporter les cartes actuelles pour obtenir un exemple de fichier csv à modifier et à importer.
Verrouiller / Mode édition
Autorise ou non l'ajout et l'édition des cartes.
